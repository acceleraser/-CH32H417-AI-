摘要

针对校园、写字楼等外卖取餐场景中存在的食物变质、柜体污染、管理低效等食品安全隐患，本项目设计并实现了一套以全端侧AI推理为核心感知能力的AI智能外卖柜系统，全部视觉识别任务100%在嵌入式终端MCU本地完成，云端零AI算力、零图像处理、零视觉算法。系统以国产沁恒微电子 CH32H417 双核RISC-V微控制器为唯一计算核心，V3F主控核负责DVP图像采集、LCD显示与WiFi通信，V5F协处理核运行自主设计训练并int8量化部署的紧凑型CNN模型，在无NPU、无GPU、无FPU、仅144MHz通用RISC-V核心条件下，以约20KB模型体积、<50ms单帧推理时延、≥20FPS连续帧率，实现5种食物+空柜+轻/重污染共8类场景的全端侧实时识别，真正做到"AI推理不出芯片、图像数据不出设备、单字节编码上云"。双核通过固定地址共享内存(0x20178000)构建三阶段流水线，AI推理延迟完全隐藏；云端仅部署轻量Flask+Mosquitto承担状态机管理与MQTT消息路由，不运行任何AI，1核1G主机可支撑千级终端；前端同时支持微信小程序(Taro跨端)与Android APP。系统具备断网100%可用、端到端延迟<100ms、模型仅20KB、BOM成本极低、隐私物理级零泄露等核心优势，是TinyML在国产通用MCU上的一次高完成度工程落地。
